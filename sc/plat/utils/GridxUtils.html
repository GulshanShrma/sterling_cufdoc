GridxUtils

This utility class contains methods like adding items to grid, deleting items from grid, modifying or updating the items in grid, disabling and enabling grid,adding preview panel, hidding and showing the grid column etc.

Methods

addUnCommitedRecordToGridUsingUId
The addUnCommitedRecordToGridUsingUId method adds a record to the grid and does not update the grid store. This method returns the index of the added row.
Arguments
NameTypeDescription
screenScreenScreen instance where grid is present
uIdStringUnique ID of the grid
newModelRecordModelJSON object for the record that needs to be added to the grid. Don't specify this if you want a empty row to be added
Returns : Number

deleteUnCommittedRecordFromGridUsingUId
The deleteUnCommittedRecordFromGridUsingUId method deletes an item from the grid and does not update the grid store.
Arguments
NameTypeDescription
screenScreenScreen instance where grid is present
uIdStringUnique ID of the grid
modelItemModelModel object of the grid row that needs to be deleted

updateRecordToGridUsingUId
The updateRecordToGridUsingUId method updates the old model item of a grid record with the new one passed.
Arguments
NameTypeDescription
screenScreenThe screen instance containing the grid
uIdStringThe unique identifier of the grid
oldModelItemModelModel Item that needs to be updated
newModelItemModelNew Model Json with the values that needs to be updated
isPartialUpdateBooleanThis flag indicates whether an update is a partial update (i.e. subset of the record item) or not
skipRefreshGridBooleanThis flag should be passed as true if refresh is not required for the grid

The hideTableColumn method hides the specified column in the grid.
Arguments
NameTypeDescription
