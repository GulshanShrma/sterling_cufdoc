RepeatingPanelUtils

This utility class contains methods related to pagination for repeating panels.

Methods

startPaginationUsingUId
The startPaginationUsingUId method calls the paginated behavior mashup API and paints the paginated repeating panels.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
namespaceStringThe source namespace for which the data will be set for the repeating panels
inputDataModelThe input data to be sent to the pagination API
mashupRefIdStringThe unique Id of the paginated behavior mashup API (defaults to the value from binding data)

setModelForIndividualRepeatingPanel
The setModelForIndividualRepeatingPanel method sets the model data for a particular repeating panel.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
uniquePanelUIdStringThe unique identifier (used by the application) of the repeating panel for which data needs to be set
namespaceStringThe source namespace for which the data needs to be set for the repeating panel
modelObjModelThe model object that is used to set data for the repeating panel
optionsBeanAny extra options needed to set data for the repeating panel

returnUIdOfIndividualRepeatingPanel
The returnUIdOfIndividualRepeatingPanel method returns the unique identifier of a particular repeating panel.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
uniquePanelUIdStringThe unique identifier (used by the application) of the repeating panel whose uId attribute needs to be returned
Returns : String

hideDesktopPaginationBar
The hideDesktopPaginationBar method hides the pagination bar of a repeating panel screen in a desktop environment.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
skipOwnerScreenResizeBooleanOptional argument that should be passed as true to avoid screen resize

showDesktopPaginationBar
The showDesktopPaginationBar method shows the pagination bar of a repeating panel screen (if it is hidden already) in a desktop environment.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
skipOwnerScreenResizeBooleanOptional argument that should be passed as true to avoid screen resize
classToAddStringOptional argument that contains the CSS class to be added to the pagination bar widget

hideMobilePaginationPreviousButton
The hideMobilePaginationPreviousButton method hides the Previous button present inside the pagination bar of a repeating panel screen in a mobile environment.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
skipOwnerScreenResizeBooleanOptional argument that should be passed as true to avoid screen resize

showMobilePaginationPreviousButton
The showMobilePaginationPreviousButton method shows the Previous button present inside the pagination bar of a repeating panel screen (if it is hidden already) in a mobile environment.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels.
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
skipOwnerScreenResizeBooleanOptional argument that should be passed as true to avoid screen resize
classToAddStringOptional argument that contains the CSS class to be added to the previous button of the pagination bar widget

hideMobilePaginationNextButton
The hideMobilePaginationNextButton method hides the Next button present inside the pagination bar of a repeating panel screen in a mobile environment.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panel.
skipOwnerScreenResizeBooleanOptional argument that should be passed as true to avoid screen resize

showMobilePaginationNextButton
The showMobilePaginationNextButton method shows the Next button present inside the pagination bar of a repeating panel screen (if it is hidden already) in a mobile environment.
Arguments
NameTypeDescription

repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
skipOwnerScreenResizeBooleanOptional argument that should be passed as true to avoid screen resize
classToAddStringOptional argument that contains the CSS class to be added to the next button of the pagination bar widget

getLastPageAction
The getLastPageAction method returns the last page action done by the user for the paginated repeating panels. Default value returned is start. Other possible return values are next, previous and pageSizeChange.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
Returns : String

getLastPageActionWidget
The getLastPageActionWidget method returns the last page action widget. If the mobile app is used, either the previous or the next button widget is returned. In case of desktop, the pagination bar widget that contains the previous, next and the combo box is returned.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
Returns : Widget

scrollToRepeatingRecordNumber
Method to scroll to the required record index among the list of repeating records. Counting starts with a non-zero index.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
recordNumberNumberThe repeating panel to scroll to. The index begins with a non-zero number

setIgnoreScrollToNextPage
Setter method that enables users to ignore scroll or swipe actions to load next page. Passing ignoreScroll as true enables this behavior. To allow scroll or swipe actions to load next page, pass the parameter as false.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the repeating panels
repeatingPanelUIdStringThe unique identifier of the content pane that contains the repeating panels
ignoreScrollBooleanPass true to ignore scroll or swipe action to load next page. Pass false to retain the default behavior of loading next page on scroll or swipe
