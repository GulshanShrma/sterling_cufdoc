BaseUtils

This utility class contains methods like creating new instance of Array or Model , numeric operations like greater or less than or number in range, Date Operations, String operations, Array operations, empty check etc.

Methods

getScreenFromEventArguments
The getScreenFromEventArguments method returns the screen object from the event argument object passed.
Arguments
NameTypeDescription
eventArgumentObjModelA business object from any local event subscriber
Returns : Screen

convertToServerFormat
The convertToServerFormat method formats a Date object as a string according to a subset of the ISO-8601 standard.
Arguments
NameTypeDescription
dateDateA Date object
typeStringInput type. Valid values are 'DATE', 'TIME', 'DATETIME'
Returns : String

mergeModel
The mergeModel method merges the source model into the destination model and returns the destination model. When both models have the same data value, the source model's value is used.
Arguments
NameTypeDescription
destinationObjectModelThe destination model
sourceObjectModelThe source model
ignoreEmptyBooleanBoolean to indicate if the empty attributes inside the model object need to be ignored while copying
Returns : Model

removeBlankAttributes
The removeBlankAttributes method checks and deletes the blank attributes from the JSON Object passed as input.
Arguments
NameTypeDescription
jsonObjectModelThe JSON object from which blank attributes need to be removed

Arguments

The cloneModel method returns the clone object of the passed JSON object.
Arguments
NameTypeDescription

formatNumber
The formatNumber method returns formatted value of the passed number. If scuiDataType is passed, that is used first for finding the data type. Otherwise binding path is used. If no data type is given, only formatting is done.
Arguments
NameTypeDescription
dataNumberThe numeric value that needs to be formatted
bindingPathStringBinding path to find the data type
scuiDataTypeStringscuiDataType to find the data type
Returns : String
