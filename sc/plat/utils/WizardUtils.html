WizardUtils

This utility class contains methods related to Wizards.

Methods

nextScreen
The nextScreen method returns the next screen instance of the wizard instance.
Arguments
NameTypeDescription
wizardInstanceWizardA wizard instance
Returns : Screen

showNextScreenWithUpdate
The showNextScreenWithUpdate method invokes behavior mashup references along with next page in a single server request. The behavior mashup references are invoked first and then the init controller of the next page is invoked. This method offers better performance when save and next functionality is required for navigation between wizard pages.
Arguments
NameTypeDescription
wizardInstanceWizardA wizard instance
mashupsDataModelThe data that needs to be passed to the mashup
Returns : Screen

previousScreen
The previousScreen method returns the previous screen instance of the wizard instance.
Arguments
NameTypeDescription
wizardInstanceWizardA wizard instance
Returns : Screen

openWizardInEditor
The openWizardInEditor method opens the wizard given by wizardId in an Editor.
Arguments
NameTypeDescription
wizardIdStringUnique ID of the wizard
wizardInputModelInitial input object to wizard
editorIdStringID of the editor to place the wizard in
editorConfigModelOptional argument with the configutration parameters associated with the editor in which screen or wizard will be opened
oldScreenScreenOptional argument with the old screen instance

getCurrentPage
The getCurrentPage method returns the current screen instance of a wizard.
Arguments
NameTypeDescription
wizardInstanceWizardA wizard instance
Returns : Screen

getCurrentPageByUId
The getCurrentPageByUId method returns the uId attribute of the wizard's current screen instance.
Arguments
NameTypeDescription
wizardInstanceWizardA wizard instance
Returns : String

isCurrentPageFirstEntity
The isCurrentPageFirstEntity method queries if the page displayed in the wizard stack container is the first entity of the wizard. Returns true if it is the first entity, false otherwise.
Arguments
NameTypeDescription
wizardInstanceWizardA wizard instance
Returns : Boolean

isCurrentPageLastEntity
The isCurrentPageLastEntity method queries if the page displayed in the wizard stack container is the last entity of the wizard. Returns true if it is the last entity, false otherwise.
Arguments
NameTypeDescription
wizardInstanceWizardA wizard instance
Returns : Boolean

confirmWizard
The confirmWizard method invokes confirm on a wizard instance. As part of the method invocation, beforeConfrm and afterConfirm events are raised.
Arguments
NameTypeDescription
wizardInstanceWizardA wizard instance

closeWizard
The closeWizard method fires beforeClose event and then closes the wizard.
Arguments
NameTypeDescription
wizardInstanceWizardA wizard instance

showPreviousPage
The showPreviousPage method takes the user to previous page with provided screen type.
Arguments
NameTypeDescription
wizardInstanceWizardA wizard instance
screenTypeStringOptional argument which shows previous screen with matching screen type. Previous screen is shown if this value is not passed

NameTypeDescription
