EditorUtils

This utility class contains methods related to editor instances.

Methods

getScreenInstance
The getScreenInstance method returns the screen instance inside the editor.
Arguments
NameTypeDescription
editorEditorAn editor instance
Returns : Screen

getCurrentEditor
The getCurrentEditor method returns the selected and active editor in the application tab container.
Arguments
NameTypeDescription
Returns : Editor

focusOnEditor
The focusOnEditor method puts focus on a particular editor.
Arguments
NameTypeDescription
editorInstanceEditorAn editor instance that needs focus
editorInputModelOptional argument that contains initial input data passed to editor

refresh
The refresh method refreshes the editor passed as input. If no input is passed, the current editor is refreshed.
Arguments
NameTypeDescription
editorInstanceEditorAn editor instance that needs to be refreshed. Optional.
