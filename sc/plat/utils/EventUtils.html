EventUtils

This utility class contains methods related to Event Management

Methods

fireEventToChild
The fireEventToChild method publishes an event to the child screen from the parent screen.
Arguments
NameTypeDescription
screenScreenScreen instance to fire the event
childScreenUIdStringUnique id of child screen inside the parent screen
eventNameStringName of the event
eventDefinitionBeanObject passed to the subscribers through the event
Returns : Bean

fireEventToParent
The fireEventToParent method publishes an event to the parent screen.
Arguments
NameTypeDescription
screenScreenscreen instance to fire the event
eventNameStringname of event
eventDefinitionBeanobject passed to subscribers from event
Returns : Bean

stopEvent
The stopEvent method stops an event. Any subsequent subscriber of the event will not get executed and if any subscriber is blocked, it'll be ignored.
Arguments
NameTypeDescription
bEventBusinessEventEvent object pertaining to the event to be stopped
