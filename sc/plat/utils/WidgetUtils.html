WidgetUtils

This utility class contains methods like hiding/showing the Widget, disabling/enabling the Widget, adding/removing/toggling the css classes, marking/clearing error widget etc.

Methods

hideWidget
The hideWidget method hides the widget associated with uId passed.
Arguments
NameTypeDescription
screenScreenA screen instance
uIdStringUnique ID of the widget to be hidden
skipOwnerScreenResizeBooleanOptional argument with value as true to avoid screen resize(Defaults to false)

showWidget
The showWidget method shows the widget associated with uId passed.
Arguments
NameTypeDescription
screenScreenA screen instance
uIdStringUnique ID of the widget to be shown
skipOwnerScreenResizeBooleanOptional argument with value as true to avoid screen resize (Defaults to false)
classToAddStringOptional argument with the CSS class to be added to the widget when it is shown

closePopup
The closePopup method closes a pop-up.
Arguments
NameTypeDescription
screenScreenA screen instance inside the pop-up/dialog
actionPerformedStringOptional
preserveDialogBooleanOptional argument with value as true to hide and false to destroy the pop-up (Defaults to false)

markFieldinError
The markFieldinError method marks the widget as an error field.
Arguments
NameTypeDescription
screenScreenA screen instance
uIdStringUnique ID of the widget
errorMessageStringError message which needs to be displayed. If this is not passed, default error message associated with the widget will be used
skipFocusBooleanIf this flag is true, focus is not set on the widget.Otherwise, it is set

clearFieldinError
The clearFieldinError method clears the error condition of a given widget.
Arguments
NameTypeDescription
screenScreenA screen instance
uIdStringUnique ID of the widget

destroyWidget
The destroyWidget method destroys the widget identified by the uId passed to this method.
Arguments
NameTypeDescription
screenScreenA screen instance
uIdStringUnique ID of the widget

setLinkImageSrc
The setLinkImageSrc method changes the image inside a link dynamically.
Arguments
NameTypeDescription
screenScreenThe screen instance that contains the Link
uIdStringThe uId of the Link to be modified
imageSrcStringThe path of the new image to be rendered. It may or may not be relative to the context root and depends on the appendToContextRoot attribute on the link

createIframeWidget
The createIframeWidget method creates an Iframe widgets dynamically in the screen and returns the UId of the dynamically created widget.
Arguments
NameTypeDescription
screenScreenThe screen instance in which the widget is to be created
uIdStringThe uId of the new widget to be created. If passed as null, auto generated uId is returned
configObjBeanThe JSON object that contains all the attributes such as frameId, frameSrc, applyCSRF, and others required to create an IFrame
Returns : String

placeAt
The placeAt method places a dynamically created widget with respect to another existing widget on the screen.
Arguments
NameTypeDescription
screenScreenThe screen instance in which the new widget is to be placed
refUIdStringThe uId of the widget with respect to which the new widget needs to be placed. This is the reference widget
createdUIdStringThe uId of the dynamically created widget that needs to be placed on the screen
positionStringSpecifies where to place the new widget with respect to the reference widget. Supported values: first, last, before or after. NOTE: first and last should only be used when reference widget is a container

setIFrameSrc
The setIFrameSrc method updates the frameSrc attribute for an IFrame widget with a given uId. IFrame widget will automatically re-load to reflect the updated frameSrc.
Arguments
NameTypeDescription
screenInstanceScreenThe instance of the screen that contains the IFrame widget
iframeUIdStringThe uId of the IFrame widget to update
srcStringMandatory. The new value of the frameSrc attribute of for the widget. The IFrame widget will automatically re-load to point to the new src
qryParamsListListOptional argument with the qryParamsList attribute for the IFrame widget

changeImageSrc
The changeImageSrc method changes the src attribute of an image.
Arguments
NameTypeDescription
screenScreenA screen instance
uIdStringUnique ID of the image widget
imageSrcValueStringNew value of the image src attribute
appendToContextRootBooleanIf the value is true, the imageSrcValue will be appended to the current application context root

changeImageAlt
The changeImageAlt method changes the alt attribute of an image.
Arguments
NameTypeDescription
screenScreenA screen instance
uIdStringUnique ID of the image widget
imageAltValueStringNew value of the image alt attribute

changeImageTitle
The changeImageTitle method changes the title attribute of an image.
Arguments
NameTypeDescription
screenScreenA screen instance
uIdStringUnique ID of the image widget
imageTitleValueStringNew value of the image title attribute
