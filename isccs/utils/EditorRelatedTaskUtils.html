EditorRelatedTaskUtils

This utility class contains methods related to the related task panel in the application.

Methods

showOrHideRelatedTaskScreen
The showOrHideRelatedTaskScreen method shows (if hidden) or hides (if shown) the related task panel in the screen. The method showRelatedTaskScreenHolder should be called to show the related task panel holder before calling this method.
Arguments
NameTypeDescription
editorScreenThe editor instance for which related task panel needs to be shown or hidden.

showRelatedTaskScreenHolder
The showRelatedTaskScreenHolder method shows the related task panel holder in the screen. Related task panel holder should be shown before calling showOrHideRelatedTaskScreen method to show the related task screen.
Arguments
NameTypeDescription
editorScreenThe editor instance for which related task panel holder needs to be shown.
