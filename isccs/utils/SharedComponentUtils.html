SharedComponentUtils

This utility class contains methods that are commonly used across the application. These methods can also be used while extending the application.

Methods

openAddressCapture
The openAddressCapture method opens the address capture screen in a popup window.
Arguments
NameTypeDescription
screenScreenScreen from which the pop-up window is opened.
screenInputModelInitial input to the screen.
personInfoModelPersonInfo object needed to load the screen. Pass empty model if it is a new address. If the values are driven from common codes, please pass the CodeValue instead of passing the CodeShortDescription.
enterpriseCodeStringEnterprise code of the current or selected enterprise.
popupTitleStringTitle which must be shown in the pop-up window.
callBackHandlerStringFunction or method name in the base screen where processing must be done after the data is saved. This call back function gets the following attribute - 1. actionPerformed: This is a string attribute which informs what was the action performed on the address capture pop-up. The values can be APPLY or CLOSE. 2. model - This is the model object which is returned by the pop-up window. 3. popupParams - This is a bean object which contains the parameters which were passed on the pop-up such as binding object, isModal, screenId, screen Input, title.
