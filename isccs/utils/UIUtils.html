UIUtils

This utility class contains methods which are used for user interface handling of the application.

Methods

callApi
The callApi method expedites calling an api by accepting a screen, input model and mashup reference id.
Arguments
NameTypeDescription
screenScreenScreen from which you want to invoke a mashup
modelInputModelThe input object to the API.
mashupRefIdStringThe mashup reference identifier.
mashupContextBeanThis is the mashup context object. If it is not passed, ControllerUtils.getMashupContext will be used to fetch the context object

callApis
The callApis method expedites calling multiple apis by accepting a screen, a list of input models and a corresponding list of mashup reference ids.
Arguments
NameTypeDescription
screenScreenScreen from which you want to invoke a mashup
modelInputListArray of input objects to the APIs.
mashupRefIdListThe mashup reference identifiers to the APIs.
mashupContextBeanThis is the mashup context object.If it is not passed, ControllerUtils.getMashupContext will be used to fetch the context object
optionsBeanBeanThis is the options object in the callMashups.This object can take options like aggregate=true, If it is not passed, ControllerUtils will use the default configurations.

openSimplePopup
The openSimplePopup method opens a popup with standard parameters.
Arguments
NameTypeDescription
screenNameStringThe complete path of the screen which needs to be opened as pop-up.
titleStringTitle which needs to be shown for the screen.
screenScreenScreen from where the pop-up needs to be opened.
popupParamsBeanAny extra information for the pop-up screen can be passed here. For eg : Create an object called as screenInput inside the popupParams to pass the input to the screen.
optionsBeanInformation about call back handlers needed for closing the pop-up can be passed here. Pass the call back handler in closeCallBackHandler object.

startPagination
This method takes the grid id, input data and options and starts the pagination in a grid by calling the corresponding mashup. Once the results are retrieved, the results are populated on the grid and the pagingation bar is set appropriately.
Arguments
NameTypeDescription
screenScreenScreen which contains the paginated grid.
gridIdStringUnique identifier of the grid.
inputDataModelInput model to the pagination mashup call.
optionsBeanA bean object with the following properties - mashupRefId: contains the mashupRefId to make the pagination call.

logoutUser
Logs out the user.
Arguments
NameTypeDescription

compareAddressObjects
Used to compare two address objects
Arguments
NameTypeDescription

AddressObject2ModelModel containing address object to be compared.
