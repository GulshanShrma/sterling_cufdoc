BaseTemplateUtils

This utility class contains methods which are used for updating title, subtitle and customer messages in the application.

Methods

updateTitle
This updateTitle method sets the title and subtitle on the current editor.
Arguments
NameTypeDescription
screenScreenScreen instance which is updating the title.
titleStringTitle that needs to be updated on the current editor.
subTitleStringSubtitle that needs to be updated on the current editor.

updateCustomerMessage
The updateCustomerMessage method updates the message on the customer message panel.
Arguments
NameTypeDescription
screenScreenScreen instance which is updating the customer message panel.
messageStringThe message that needs to be updated on the customer message panel.
localizeBooleanIf this argument is true, translated bundle value of the message is updated. If false, message is updated on the customer message panel as is.
