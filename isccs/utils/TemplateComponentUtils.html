TemplateComponentUtils

This utility class contains methods which are used for handling search screens in a pop-up window.

Methods

openUserLookup
The openUserLookup method gets the targetModel for the widget and opens the User Search screen in a pop-up.
Arguments
NameTypeDescription
eventUIEventUI Event object associated with the event.
bEventBusinessEventBusiness event object associated with the event.
ctrlStringThe uId of the text field from where the search popup needs to be opened.
argsBeanArgs contains any extra parameters which can be passed to the User Search screen. The attributes which are passed are originatingControlInstance, which is the originating widget and screen instance where the originating widget resides.

confirmUserLookupSelection
The confirmUserLookupSelection method handles the user lookup selection. It sets the model on the parent screen to populate fields accordingly.
Arguments
NameTypeDescription
screenScreenThe current screen.
actionStringThe action performed on the screen like APPLY, CLOSE, and so on.
outputModelThe model object that contains the output from the pop-up window.
paramsBeanThe params object contains parameters sent by the pop-up window, if any. The attributes which are passed are originatingControlInstance, which is the originating widget and screen instance where the originating widget resides.
ctrlStringThe originating control that opened the pop-up window.
