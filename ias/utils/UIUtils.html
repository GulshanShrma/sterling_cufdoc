UIUtils

This utility class contains common methods for UI rendering.

Methods

callApi
The callApi method calls an API.
Arguments
NameTypeDescription
screenScreenScreen instance that needs to invoke a mashup.
modelInputModelInput model object to the API.
mashupRefIdStringMashup reference identifier.
mashupContextBeanMashup context object. If it is not passed, ControllerUtils.getMashupContext will be used to fetch the context object.

callApis
The callApis method calls multiple APIs.
Arguments
NameTypeDescription
screenScreenScreen instance that needs to invoke a mashups.
modelInputListArray of input model objects to the APIs.
mashupRefIdListArray of mashup reference identifier.
mashupContextBeanMashup context object. If it is not passed, ControllerUtils.getMashupContext will be used to fetch the context object.
optionsBeanBeanOptions object in the callMashups. This object can take options like aggregate=true, ff it is not passed, ControllerUtils will use the default configurations.

openSimplePopup
The openSimplePopup method opens a screen as a pop-up.
Arguments
NameTypeDescription
screenNameStringComplete path of the screen which needs to be opened as pop-up.
titleStringTitle which needs to be shown for the screen.
screenScreenScreen from where the pop-up needs to be opened.
popupParamsBeanAny extra information for the pop-up screen can be passed here. For eg : Create an object called as screenInput inside the popupParams to pass the input to the screen.
optionsBeanInformation about call back handlers needed for closing the pop-up can be passed here. Pass the call back handler in closeCallBackHandler object.

openWizardInEditor
The openWizardInEditor method opens the wizard in a new editor. This method reads the EditorConfig.json file and determines the editor to be used in mobile environment.
Arguments
NameTypeDescription
wizardIdStringWizard Id of the wizard to open.
modelDataModelInput model to the wizard.
editorIDStringEditor Id of the editor in which to open the wizard.
oldScreenScreenOld screen instance.
customEditorConfigBeanEditor config parameters.

logoutUser
The logoutUser method logs out the user.
Arguments
NameTypeDescription
