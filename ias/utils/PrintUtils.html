PrintUtils

This utility class contains common methods for print related tasks.

Methods

printHtmlOutput
The printHtmlOutput method displays the HTML output in an iframe or pop-up window for printing, based on the browser or device.
Arguments
NameTypeDescription
outputModelModelOutput model used to paint the print html layout.

printHtmlInIframe
The printHtmlInIframe method appends the HTML output to the iframe and invokes the browser's print command.
Arguments
NameTypeDescription
outputModelModelOutput model used to paint the print html layout.

printHtmlInPopup
The printHtmlInPopup method appends the HTML output to the pop-up window.
Arguments
NameTypeDescription
outputModelModelOutput model used to paint the print html layout.
