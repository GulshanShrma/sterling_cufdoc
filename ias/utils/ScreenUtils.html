ScreenUtils

This utility class contains common methods for screen handling.

Methods

scrollTop
The scrollTop method scrolls the specified widget to top and into the visible area below the fixed header.
Arguments
NameTypeDescription
screenScreenScreen instance which has the widget to be scrolled to
widgetUIdStringWidget UId
offsetStringExtra offset from above fixed height

scrollTopWithFixedHeader
The scrollTop method scrolls the specified widget to top and into the visible area below the fixed header.
Arguments
NameTypeDescription
screenScreenScreen instance which has the widget to be scrolled to
widgetUIdStringWidget UId
offsetStringExtra offset from above fixed height
fixedHeightStringFxied height which is not scrollable.

scrollToWidgetTop
The scrollToWidgetTop method scrolls the specified widget to top and into the visible area below the fixed header. This method should be used as event subscriber handler.
Arguments
NameTypeDescription
eventUIEventUI event object associated with the event.
bEventBusinessEventBusiness event object associated with the event.
ctrlStringUId of the control widget.
argsBeanEvent arguments contains any extra parameters to be passed for the even handler.

scrollToWidgetAfterPageLoad
The scrollToWidgetAfterPageLoad method scrolls the repeating screen widget to top or scrolls to the pagination bar and into the visible area below the fixed header. This method should be used as event subscriber handler.
Arguments
NameTypeDescription
eventUIEventUI event object associated with the event.
bEventBusinessEventBusiness event object associated with the event.
ctrlStringUId of the control widget.
argsBeanEvent arguments contains any extra parameters to be passed for the even handler.

Arguments
NameTypeDescription

optionsModelOptions passed as the model.
dataValueStringData value associated with the binding.
Returns : String

getForwardBindingNamespaceMapBean
The getForwardBindingNamespaceMapBean method returns the constructed forward-binding namespace-map.
Arguments
NameTypeDescription
parentnamespaceStringName of the parent screen's namespace.
parentpathStringPath of the parent screen's model data to be forwarded.
childnamespaceStringName of the child screen's namespace.
childpathStringPath of the child screen's model to set model data.
Returns : Bean
