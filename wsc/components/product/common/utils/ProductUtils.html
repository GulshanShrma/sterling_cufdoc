ProductUtils

This utility class contains methods to display product information.

Methods

createItemVariationPanel
The createItemVariationPanel method creates the 'LabelSetWidget' widget to display the attributes of a product variant.
Arguments
NameTypeDescription
screenScreenScreen instance of the screen in which the Product Variation panel needs to be created.
uIdStringUId of the content pane where the product variation attributes needs to be painted.
modelModelInput model used to create Product Variation panel.
attributePathStringPath to variation attribute in the input model
Returns : Widget

openItemDetails
The openItemDetails method opens the "Product Details" screen.
Arguments
NameTypeDescription
screenScreenScreen instance from where the "Product Details" screen needs to opened.
itemDetailsModelModelInput model to open product details screen.
