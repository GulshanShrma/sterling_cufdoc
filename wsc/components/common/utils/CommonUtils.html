CommonUtils

This utility class contains generic methods that can be used across the store application.

Methods

getDisplayOrderNumber
The getDisplayOrderNumber method returns either the order number, comma separated order numbers or 'Multiple'.
Arguments
NameTypeDescription
orderNoStringOrderno string which needs to displayed.
commasOrLiteralStringAllowed values 'COMMAS|LITERAL'. 'COMMAS' returns comma separated order no. 'LITERAL' returns 'Multiple'.
Returns : String

getBundleKeyForTabHeader
The getBundleKeyForTabHeader method returns either the order number or 'Multiple' to display in tab headers.
Arguments
NameTypeDescription
orderNoStringOrderno string which needs to displayed.
Returns : String
